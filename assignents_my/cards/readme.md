# App Structure

## Black Jack

### Buttons 
- Deal
  - gives card
  - if player has 21 - player win
  - if player has > 21 - player lose
- Stop
  - check if player cards are < 21
  - deal 2 cards to house 
  - if house closer to 21 - house wins
  - if not (less than player or more than 21) - player win
